## Description

Admin backend for a shopping store

## Installation of dependencies

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```

## Migrations

#### Create migrations 
```bash
yarn migrate:create ./src/database/migrations/MigrationName
```

#### Run migrations
```bash
yarn migrate:run
```

#### Revert migrations
```bash
yarn migrate:revert
```
